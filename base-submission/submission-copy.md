# Base Collect Stamp

## App Name

Base Collect Stamp

## Tagline

Fill your Base stamp board

## Description

Claim themed stamps on Base and complete a collectible progress board built for repeat visits.

Character count: 92

## Category

Collectible / progression / onchain utility

## Short Pitch

Base Collect Stamp turns simple wallet actions into a repeatable collection loop, letting users unlock themed marks and fill a shared progress board.

## Production Environment Variables

- `NEXT_PUBLIC_COLLECT_STAMP_CONTRACT_ADDRESS`: deployed stamp contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
